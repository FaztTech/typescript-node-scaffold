## Requirements
    - Node.js
    - Typecsript
        
## Commands
    - to install typescript: `npm install -D typescript`
    - to install @types: `npm install -D @types/<package>`
        - for node: `@types/node`
    - to init a config: `tsc --init`
        - outDir
        - target
        - module
        - include / exclude
